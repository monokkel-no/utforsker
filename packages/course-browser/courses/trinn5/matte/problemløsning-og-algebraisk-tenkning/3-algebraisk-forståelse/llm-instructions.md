# 3 Algebraisk forståelse

## 3.1 Likhet og balanse

Forstå likhetstegnet som "samme verdi på begge sider"

Løse enkle likninger (□ + 6 = 14)

Finne ukjent i multiplikasjon og divisjon

## 3.2 Representasjon med symboler

Bruke symbol for ukjent verdi

Lage og tolke enkle regneuttrykk

Oversette tekst til algebraisk form

## 3.3 Sammenhenger mellom størrelser

Utforske proporsjonale situasjoner (pris per stykk × antall)

Lage enkle formler

Bruke tabell for å vise sammenheng mellom variabler

## 3.4 Regnestruktur

Forstå regnerekkefølge (på grunnleggende nivå)

Se struktur i sammensatte uttrykk

Bruke parenteser i enkle uttrykk

## 🎯 Mestringsnivå (tverrgående progresjon)

Kompetansen kan utvikles gjennom nivåer:

Gjenkjenne og utføre (med støtte)

Forklare og anvende selvstendig

Generalisere og overføre til nye situasjoner
